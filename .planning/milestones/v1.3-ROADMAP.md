# Milestone v1.3: Quota Enforcement & API Key Management

**Status:** ✅ SHIPPED 2026-05-25
**Phases:** 15–17
**Total Plans:** 14

## Overview

Harden the cloud platform with real enforcement teeth — 429 hard caps at ingest across all four quota dimensions (API calls, tokens, spend, seats), the full API key lifecycle surfaced in the UI with a viewer-creator scoping model, and the v1.2 Google routing gap closed by extending model downgrade beyond body rewrite into URL-path rewrite. Phase 18 (Usage Dashboard Improvements / DASH-01–04) was originally scoped to v1.3 and has been deferred to v1.4.

## Milestone Summary

**Key Decisions:**
- Hard enforcement is server-side only — local proxy stays unmetered (free forever per core principle)
- `_check_quota_or_raise()` is called before any DB write at POST /v1/ingest — fail-closed on quota exceeded
- `QuotaExceededDetail` response body is structured Pydantic JSON, not free text — clients can parse `dimension/current/limit` programmatically
- D-04 viewer-creator scoping: viewers can self-create/self-revoke their own keys; cross-creator access returns 404 (indistinguishability), not 403
- Resend-verification reads identity from server-side JWT session — never from client-supplied email — closes AUTH-08 and prevents enumeration regressions
- ROUTE-08 implemented via polymorphic `rewrite_path_for_routing()` Provider hook so future providers can add path rewriting without core changes
- DOWNGRADE_MAP keys normalized to suffix form (`models/gemini-1.5-flash` → `gemini-1.5-flash`) so URL-path matching is single-source-of-truth

**Issues Resolved:**
- v1.2 known limitation closed: Google routing now downgrades via URL path, not just body
- CR-01, CR-02, CR-03 all closed in Phase 16 (viewer guard, fail-open resend, truthful UI)
- SC-5 ↔ D-04 contract divergence reconciled in 16-PLAN-10

**Issues Deferred:**
- Phase 18 / DASH-01–04 — Usage Dashboard Improvements (date-range picker, model breakdown, CSV export, daily trend chart) moved to v1.4 milestone

**Technical Debt Incurred:**
